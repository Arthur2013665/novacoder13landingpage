# Nova AI - VS Code Extension

Elite AI coding assistant right in your IDE.

## Features
- **Chat Panel** — Open with `Ctrl+Shift+N` (or `Cmd+Shift+N` on Mac)
- **Explain Code** — Right-click selected code → "Nova AI: Explain Selected Code"
- **Fix Code** — Right-click selected code → "Nova AI: Fix Selected Code"  
- **Generate Tests** — Right-click selected code → "Nova AI: Generate Tests"

## Setup
1. Install the extension (Load Unpacked in VS Code dev mode, or package with `vsce`)
2. Open VS Code Settings and configure:
   - `nova.apiUrl` — Your Nova AI API endpoint
   - `nova.apiKey` — Your API key

## Keyboard Shortcuts
| Command | Shortcut |
|---------|----------|
| Open Chat | `Ctrl+Shift+N` |

## Compatible IDEs
- VS Code
- Cursor
- Kiro
- Any VS Code-based IDE
